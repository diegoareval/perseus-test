
# Perseus Test
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev

# build
$ npm run build

# production mode
$ npm run start

# run  lint
$ npm run lint

# run  fix linter
$ npm run lint:fix
```
